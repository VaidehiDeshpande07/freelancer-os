Freelancer Automation Workflow & Smart Portfolio System
-Aligned with UN SDG 8 – Decent Work & Economic Growth

🌟 Overview
This system is designed to empower independent freelancers by automating workflow processes, tracking payments, creating invoices, and building a professional smart portfolio. It helps freelancers streamline operations, improve credibility, and build a formal digital economic identity through a Freelancer Credit Score.
The platform bridges the gap between freelance work execution and structured financial empowerment, contributing to global economic inclusion.

🌍 Global Problem Solved
-1.5+ billion freelancers globally, many without structured workflow tools.
-60–80% of early freelancers lack systems for invoicing, payment tracking, and portfolio management.
-Missed follow-ups → delayed income and professional setbacks.
-No formal credit history → financial exclusion, limiting access to loans or financial products.

System Impact:
✔ Reduces administrative workload
✔ Automates reminders and follow-ups
✔ Provides digital economic identity and formal credit reliability
✔ Enhances professional branding and income transparency

💡 Key Features

Core Workflow Automation
1. User Authentication – Secure login/signup with roles
2. Smart Invoice Generator – Auto-numbered, client management, GST-ready, downloadable PDFs
3. Automated Follow-Ups – Dashboard alerts, email reminders, overdue detection
4. Payment Tracking Dashboard – Total earnings, pending & overdue invoices, monthly income graphs
5. Smart Portfolio Builder – Add projects, skill tagging, category filtering, shareable public link, resume + portfolio PDF export

Strategic Upgrade
1.Freelancer Credit Score (0–100) – Evaluates reliability based on on-time completion, payment cycle, client retention, portfolio completeness, and activity consistency
2.Impact: Provides digital economic identity → loan eligibility, BNPL tools, freelancer financing

🧠 Tech Stack
Frontend: React.js, Tailwind CSS, Chart.js, React Router
Backend: Node.js, Express.js, JWT, Nodemailer
Database: MongoDB (Hackathon), PostgreSQL (Production)
Deployment: Frontend → Vercel, Backend → Render, Database → MongoDB Atlas, Email → SendGrid
